# README

Ama Tutor

Ask-Me-Anything Tutor 

## Setup instructions

1. Checkout the repo.
2. Install Ruby 3.3.5 (using rbenv)
3. go to amatutor drive and run below

   > brew install rbenv
   
   > rbenv install 3.3.5

    > rbenv local 3.3.5

   > rbenv exec bundle install

5. Download ollama
   > ollama run llama3.1

6. Go to project directory and run
    > docker compose up.
    
    > rails g server
